# iframe与跨域通信

## 跨域
在浏览器地址栏输入一个地址时,这个地址通常包含四部分信息内容包括: 1.协议,2.域名,3.端口,4.资源位置.
其中前三部分即协议,域名,端口存在一个不一样,浏览器就会认为存在跨域的问题
> 例如: 
+ http://father.moxiao.com/a/a.html和http://father.moxiao.com/b/b.html	不存在跨域
